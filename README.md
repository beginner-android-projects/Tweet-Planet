# Tweet Planet
### This project displays Tweets using Twitter's [Filtered Streams API](https://developer.twitter.com/en/docs/twitter-api/tweets/filtered-stream/introduction) on Google Maps 

## :rocket:	Components Used
- [Android Architecture Components](https://developer.android.com/topic/libraries/architecture)
- [Navigation Components](https://developer.android.com/guide/navigation)
- [Data Binding](https://developer.android.com/topic/libraries/data-binding)
- [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)

This application shows the use of **Flow** for collecting data from Twitter's Filtered Stream API.
